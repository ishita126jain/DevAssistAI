gRPC is a high-performance Remote Procedure Call framework developed by Google.

It allows communication between microservices using HTTP/2.

gRPC uses Protocol Buffers (protobuf) for fast serialization.

Benefits of gRPC:
- Faster than REST
- Strongly typed contracts
- Supports streaming
- Cross-platform communication

In ASP.NET Core, gRPC services are created using .proto files and service contracts.

gRPC is commonly used in microservices and distributed systems.
